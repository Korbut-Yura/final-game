RULES

  1. You start the first.
  2. Select the spell (each spell is a separate task).
  3. To apply the spell, you need to complete the task.
  4. If the task was decided incorrectly, the move passes to the enemy.
  5. Win as many opponents as possible.
  6. Have fun!
